### ESLint

Visual Studio CodeでESLintを \*\*.Vue で適用させるためにはユーザー設定に下記の記載が必要。

```javascript
"eslint.validate": [
  "javascript", {
    "language": "vue",
    "autoFix": true
  }
]
```
